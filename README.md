## 📌 Project Overview

This project analyses **Energy Performance Certificate (EPC)** data for residential properties in Manchester (2014–2024).

The goal was to convert **raw, inconsistent data** into a **clean, structured dataset** and build an **interactive dashboard** to support:

* Energy efficiency analysis
* CO₂ emissions tracking
* Policy and retrofit decision-making

---

## ⚙️ End-to-End Workflow

### 1. Data Collection

* Source: UK Government EPC dataset
* Scope: Manchester residential properties
* Size: **133,000+ records with 50+ variables**

---

### 2. Data Cleaning & Transformation (SQL Server)

Key operations:

* Removed duplicate records using latest lodgement date
* Cleaned invalid CO₂ emission values
* Standardised categorical fields (e.g., property type)
* Created structured tables for analysis

#### Advanced SQL Components:

* Views (e.g., energy rating distribution)
* Stored Procedures (dynamic filtering, KPI summaries)
* Data validation checks (NULL analysis, row counts)

---

### 3. Data Modelling & Analysis (Power BI)

#### DAX Measures Created:

* Property count
* Average energy efficiency
* CO₂ emissions (current vs potential)
* Energy efficiency gap
* Percentage of properties needing improvement

#### Dashboard Features:

* KPI Cards (Total Properties, Avg Efficiency)
* Pie Chart (Energy rating distribution)
* Treemap (Property distribution)
* Bar Chart (Energy gap by property type)
* Line Chart (CO₂ trends over time)
* Map Visual (Geographic emission distribution)
* Slicers (Year, Property Type)

---

## 📊 Key Insights

* Majority of properties fall in **EPC ratings C and D**, indicating moderate efficiency
* Significant **energy improvement potential**, especially in houses
* **CO₂ emissions show a long-term declining trend** (2014–2024)
* Geographic analysis reveals **high-emission clusters**, enabling targeted interventions
* Flats show better efficiency compared to houses due to structural advantages
